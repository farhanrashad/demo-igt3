About
============
This module imports lot/serial picking from CSV/Excel files in a single click. You can transfer a lot/serial pickings from one location to other locations. When you transfer lot/serial picking, you have the option to create a lot/serial on the destination location. You can import custom fields from CSV or Excel.

Installation
============
1) Copy module files to addon folder.
2) Restart odoo service (sudo service odoo-server restart).
3) Go to your odoo instance and open apps (make sure to activate debug mode).
4) click on update app list.
5) search module name and hit install button.
